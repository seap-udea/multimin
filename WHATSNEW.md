# MultiMin

## What's New

`MultiMin` is under active development. Here you will find a list of features introduced in each version of the package.

## Version History

### 0.1.0-deprecated (2026-01-21)

**Initial Release as MultiNEAs (Deprecated)**

This was the first release, establishing the basic package structure:

- **Package Structure**: 
  - Created standard Python package layout with `src/` directory structure
  - Configured for PyPI distribution with `setup.py` and `pyproject.toml`
  - Added comprehensive `.gitignore` for Python development

- **Documentation**:
  - Initial `README.md` with project overview and installation instructions
  - `LICENSE` file with GNU Affero General Public License v3 (AGPL-3.0)
  - This `WHATSNEW.md` changelog

- **Testing Framework**:
  - Set up `tests/` directory for unit tests
  - Configured pytest as the testing framework

- **Core Module**:
  - Basic `__init__.py` with package metadata
  - Version tracking system

### Planned Features

Future versions will include:

- **0.2.x versions**:
  - Orbital mechanics utilities

- **0.3.x versions**:
